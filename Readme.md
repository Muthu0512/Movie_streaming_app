# Netflix Clone — Movie Streaming Platform

A full-stack movie streaming web application built with the MERN stack, inspired by Netflix. Fetches real-time movie and TV show data from the TMDB API with features like user authentication, trailer playback, search history, and content recommendations.

---

## Live Demo
https://movie-streaming-app-8d90.onrender.com/



## Features

- JWT Authentication — Signup, Login with protected routes
- Home Page — Dynamic Hero section with Play & More Info buttons
- Movie Categories — Trending, Popular, Top Rated, Now Playing sliders
- TV Shows — Airing Today, On the Air, Popular, Top Rated sliders
- Watch Page — Inline YouTube trailer, movie details & poster
- Recommendations — Similar Movies / TV Shows on watch page
- Search — Search Movies, TV Shows & Persons with filter toggle
- Search History — Logs queries with date, type badge & delete option
